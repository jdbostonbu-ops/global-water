# Global Water
# 💧 AquaPulse: 3D Global Water Health Tracker
An interactive 3D geospatial visualizer for global water stress and quality data.

## 🌍 The Problem
Global water data is trapped in static PDFs and academic charts. The public and policymakers lack a "living" view of our planet's most vital resource.

## 🚀 The Solution
A high-performance 3D Globe that allows users to toggle layers of satellite data (NASA/UN) to see real-time water health, pollution hotspots, and drought trends.

## 🛠 Tech Stack
- **Frontend:** Vite + React
- **Graphics:** CesiumJS (Planet-scale 3D Rendering)
- **Data Sources:** NASA Earth Data API / UN-Water API
- **State Management:** Zustand (for smooth layer toggling)

## ⚡ Key Features
- **3D Bathymetry:** Visualize underwater terrain and aquifer levels.
- **Time-Lapse:** Scrub through 10 years of pollution data to see trends.
- **Citizen Science Layer:** Real-time upload points for local water test results.
